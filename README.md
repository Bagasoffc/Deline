---

Instalasi di Termux

1. pkg upgrade -y && pkg update -y


2. git clone https://github.com/Bagasoffc/Deline


3. cd ~/Deline


4. bash install.sh


5. npm start




---

Instalasi Manual

1. pkg upgrade -y && pkg update -y


2. pkg install nodejs git ffmpeg -y


3. git clone https://github.com/Bagasoffc/Deline


4. cd Deline


5. npm install


6. npm start
